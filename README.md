# 網頁管理平臺 · 登入頁面

本頁面用於使用者的登入驗證、令牌儲存。

## 環境和準備工作要求
- `python3`
- `git submodule init`
- `npm install`

## 常用指令
- `npm run release`: 完整建立建立壓縮的釋出版本網頁，生成位於根目錄下的主頁檔案 `index.html` 。
- `npm run debug`: 生成除錯模式網頁。
- `npm run html`: 生成並不處理 JS ，用於修改 HTML 和 CSS 時快速生成。
- `npm run clean`: 清除生成出的檔案。
